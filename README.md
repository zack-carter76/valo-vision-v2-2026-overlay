## Overview

Valo-Vision-V2-2026 is a companion overlay and support tool for Valorant that runs across platforms. It brings together AI-guided agent pick assistance, utility prediction, and match analysis in one interface, giving players quick tactical context while they are in-game.

The project is also built with adaptability in mind. It includes customization options, multilingual interface support, and synced overlay behavior across supported environments. The goal is to provide a compact decision-support toolkit for use before, during, and after matches without needing to jump between separate applications.
